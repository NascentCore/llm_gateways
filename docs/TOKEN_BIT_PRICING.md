# Token 按比特计价 vs 移动流量：数量级对比

## 结论（一句话）

在**把 token 换算成比特**、并用**月租套餐里的通用流量**代表“上网比特单价”时，常见大模型 API 的**每比特名义成本**通常比移动流量**高约 \(10^3\sim10^4\) 倍**；**输出 token** 往往再贵几倍。

---

## 口径（要对齐才能比）

| 项目 | 假设/说明 |
|------|-----------|
| **Token → 比特** | 粗算：**1 token ≈ 4 byte ≈ 32 bit**（偏英文/混合文本的中位假设；中文会偏离，但数量级通常不变） |
| **移动“带宽”基准** | 用**月租档位里的 GB 流量**折算 **¥/bit**；月租常含语音/权益，**整月租全算进流量**会让“流量侧单价”偏**贵**，从而倍数偏**小**（保守下限） |
| **API 基准** | 以公开 **$/M tokens** 换算为 **$/bit**（与移动侧统一到美元仅便于同列对比，汇率按约 **7.2 CNY/USD** 粗算） |

---

## 参考档位（移动）

- **便宜流量密度**：约 **39 元 / 50GB** → 约 **0.78 元/GB**
- **较贵流量密度**：约 **128 元 / 30GB** → 约 **4.27 元/GB**

按 **1GB = \(10^9\) byte**（十进制 GB）：

\[
\text{¥/bit}=\frac{\text{¥/GB}}{8\times 10^9}
\]

---

## 参考价格（LLM，公开量级）

- **GPT-4o**：输入约 **$2.5 / 1M tokens**；输出约 **$10 / 1M tokens**（以 [OpenAI API 定价](https://platform.openai.com/docs/pricing) 为准，会随批次/缓存价变化）
- **GPT-4o mini（对照）**：输入约 **$0.15 / 1M tokens**

换算：**1M tokens ≈ 4×10⁶ byte ⇒ 3.2×10⁷ bit**（在 4 byte/token 假设下）

---

## 倍数表（LLM $/bit ÷ 移动折算 $/bit）

| 移动参考（月租全归因流量） | 对比对象 | 大约倍数 |
|---:|---|---:|
| 39 元 / 50GB | GPT-4o **输入** | **~5.6×10³** |
| 39 元 / 50GB | GPT-4o **输出** | **~2.2×10⁴** |
| 128 元 / 30GB | GPT-4o **输入** | **~1.1×10³** |
| 128 元 / 30GB | GPT-4o **输出** | **~4.2×10³** |
| 39 元 / 50GB | GPT-4o mini **输入** | **~3.4×10²** |

---

## 重要提醒（避免误读）

- **商品不同**：流量主要是**传输**；token 费用主要是**推理算力 + 软件与服务**，不是“搬比特”的等价物。
- **token ≠ 线上真实传输比特**：HTTPS/JSON/重试等还有额外流量，但**不计入 token 账单**。
- **月租 ≠ 纯比特定价**：更严谨可用**套餐外流量单价**或**纯流量卡**的 ¥/GB 重算；不同地区/渠道报价会波动，上表为**数量级**参考。

---

## 可复用的换算骨架

1. **移动**：\(\text{¥/bit}=\dfrac{\text{月租}/\text{GB}}{8\times10^9}\)（把月租全部算进 GB 时）
2. **LLM**：\(\text{$/bit}=\dfrac{\text{$/M tokens}}{3.2\times10^7}\)（在 **4 byte/token** 假设下）
3. **倍数**：\(\dfrac{\text{LLM } \$/\text{bit}}{\text{移动 } \$/\text{bit}}\)（美元侧对比时先将 ¥/bit 除以汇率）

- **非用途**：精确财务审计、合同定价、税务或运营商账单核对。
