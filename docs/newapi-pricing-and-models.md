# NewAPI 模型计费、渠道与模型名配置说明

本文汇总 [QuantumNous/new-api](https://github.com/QuantumNous/new-api)（NewAPI）中与**模型价格、配额扣费、渠道模型列表/映射、补全倍率**相关的概念与操作建议，便于与本仓库内 NewAPI 相关脚本（如 `scripts/newapi_batch_accounts/`、`test_chat.py`）对照使用。

---

## 1. 计费核心：配额（Quota）

- **配额**是系统实际**加减余额、记流水**的单位，不是美元或人民币字段。
- 文档与界面中常用换算：**约 500,000 配额 = 1 美元**（代码中 `QuotaPerUnit`），用于把「配额」和「货币」在展示、对账时对齐理解。
- **预扣 + 实扣**：请求前按估算 Token **预扣**配额，结束后按**实际 Token** 多退少补。

### 1.1 按量计费（文本，典型公式）

```
配额消耗 ≈ (输入 Token 数 + 输出 Token 数 × 补全倍率 CompletionRatio)
           × 模型倍率 ModelRatio
           × 分组倍率 GroupRatio
```

涉及**缓存读/写、图片、音频**等时，会按各类 Token 乘以对应倍率再累加，逻辑仍是「Token × 倍率 → 配额」。

---

## 2. 界面「美元 / 百万 Token」与后端倍率

- 模型计费可视化编辑里填的 **Input / Output（及缓存等）$/1M tokens** 是**给人看的定价表写法**。
- 保存时会换算为后端的 **ModelRatio、CompletionRatio**（及 Cache 等倍率）；**运行时扣费只认倍率与 Token 数**，不会每笔再用美元现场计算。
- **人民币上游**（如 MiniMax 官方「元/百万 tokens」）：需先按采用汇率换算为 **$/百万** 再填入，或按站内「元 ↔ 配额」约定反推倍率（例如与 `scripts/newapi_batch_accounts` 中额度换算口径一致）。
- 当 **输出单价 / 输入单价** 为固定比（如 MiniMax M2.7：8.4 / 2.1 = **4**）时，**补全倍率常为 4**，可避免无意义的长小数。

---

## 3. 额度展示货币（可选）

- 系统「运营/通用设置」中可将 **额度展示类型** 设为 **USD / CNY / TOKENS / CUSTOM** 等（见 `general_setting.quota_display_type`）。
- **CNY** 主要影响控制台等处**展示为人民币**（配合汇率）；**内部扣费仍以配额 × 倍率为准**。

---

## 4. 渠道：模型列表、自定义名称、模型映射

| 配置项 | 作用 |
|--------|------|
| **模型列表** | 该渠道**允许处理**的 `model` 字符串（逗号分隔存库）。用户请求中的 `model` 须能匹配到已启用能力。 |
| **自定义模型名称 +「填入」** | 仅将输入的字符串（可逗号多个）**追加进模型列表**；**不**建立与上游 ID 的映射。 |
| **模型映射** | **键** = 请求里的模型名，**值** = 转发给上游时替换成的模型名。用于对外隐藏上游真实 ID 时：列表中**只保留对外名**，映射中写 `{ "对外名": "上游真实ID" }`。 |

提交时若映射的键未出现在模型列表中，NewAPI 可能提示需先加入列表（可用「自定义模型名称」快速追加键名）。

## 5. 系统设置：模型补全倍率（CompletionRatio）与「自定义模型」

源码逻辑要点（`setting/ratio_setting/model_ratio.go`）：

- 模型名含 **`/`**（如 `minimax/minimax-m2.7`、`moonshot/kimi-v1-32k`）时，会**优先**使用「模型补全倍率」JSON 中**与该 key 完全一致**的项。
- 若**未**在 JSON 中配置，内置规则多为 `gpt-*`、`claude-*`、`gemini-*` 等**裸名**；**带厂商前缀的路径型名称**往往落回默认补全倍率 **1**（输出与输入同权），易与官网「输出更贵」不一致。

因此后台文案中的**「仅对自定义模型生效」**可理解为：**尤其需要为「内置补全规则覆盖不到或带 `/` 的实际请求模型名」在 JSON 中单独写明补全倍率**；JSON 的 **key 必须与客户端/网关实际请求的 `model` 字符串一致**（含大小写、前缀）。

部分官方裸名模型存在 **locked** 内置补全倍率，一般**不以** JSON 覆盖。
